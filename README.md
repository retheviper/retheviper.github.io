# TL;DR

## 日本語

記録用の技術ブログです。

`Hugo` をベースに、`hugo-theme-stack` テーマとカスタムレイアウト・設定を組み合わせて実装しています。

また、ポスティングは日本語を原文として作成し、必要に応じて韓国語と英語の翻訳も提供しています。

## 한국어

기술을 기록하기 위한 블로그입니다.

`Hugo`를 기반으로 `hugo-theme-stack` 테마와 커스텀 레이아웃, 설정을 조합해 구현했습니다.

또한 포스팅은 일본어를 원문으로 작성하고, 필요에 따라 한국어와 영어 번역도 함께 제공합니다.

## English

This is a technical blog for keeping records.

It is built with `Hugo`, using the `hugo-theme-stack` theme along with custom layouts and configuration.

Posts are originally written in Japanese, and Korean and English translations are provided when needed.
